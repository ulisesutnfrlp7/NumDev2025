Resumen

Este proyecto usa las siguientes librerías y convenciones:

  • Chart.js — visualización de gráficos (scatter y líneas de ajuste).
  • chartjs-plugin-zoom — paneo y zoom interactivo.
  • xlsx (SheetJS) — lectura del archivo datos.xlsx para extraer filas con columnas neighborhood, area_m2, price_kUSD.
  • numeric.js — solución de sistemas lineales (se usa para el ajuste cuadrático con sistema 3x3).
  • calculos.js — funciones matemáticas implementadas:
    • linealFit — mínimos cuadrados para y = m x + b.
    • cuadraticoFit — ajuste polinomial de grado 2 via sistema normal (resuelto con numeric.solve).
    • exponencialFit — regresión exponencial y = a e^{b x} usando log transform en y.
    • potencialFit — regresión potencial y = a x^{b} usando log-log.
    • calcularR2 — coeficiente de determinación R² calculado sobre las predicciones del modelo.
  • Convenciones:
    • Se espera que datos.xlsx tenga encabezados: neighborhood ('A','B','C'), area_m2 (numérico), price_kUSD (numérico).
    • Las páginas son multipágina por simplicidad y escalabilidad; cada una carga las mismas utilidades JS compartidas.
